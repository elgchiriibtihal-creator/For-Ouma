# 🌙 Ouma's Midnight World 💜

> A soft, purple, interactive night made specifically for **Ouma / 3wiwra**.
> Calm ✨ · Playful 😄 · Intimate 💜 · Slightly funny

---

## 🎯 Project Goal

A beautiful interactive mini-world / mini-game static website personalized for Ouma. Features a purple midnight aesthetic with intimate messages, playful interactions, and supportive moments.

---

## ✅ Completed Features

### 🌌 Intro Splash Screen
- Animated moon entrance with glowing title
- "Enter your world" button to dismiss the splash
- Starts ambient sound on entry

### 🚗 Landing Section
- Typewriter effect: "Hey Ouma… I made you a little world."
- Animated cars driving across a neon road
- Hint text: "Click a car, 3wiwra 👀"
- Scroll indicator after intro completes

### 🚗 Cars Interaction Section
- 13 clickable car cards with different vibes: soft / funny / cute / deep / special
- Names used intentionally: Ouma (soft), 3wiwra (funny/playful), Oumaima (rare/deep)
- One special car triggers a unique golden-glow message
- Beautiful popup overlay with sparkles

### 💜 Press If Sad Section
- Large soft button: "Press if Ouma is not okay 💜"
- Random comforting message from 6 options
- Floating hearts (💜✨🫶🌙) animation
- Screen glow effect
- Subtle funny sub-text: "even when you're 3wiwra and dramatic 😌"

### 💌 Floating Letters Section
- 3 floating animated envelopes
- "for Ouma" — letter of pride
- "for 3wiwra only ⚠️" — playful teasing
- "for the quiet Ouma" — strength & support
- Opens a paper-styled letter modal on click

### 🌙 Night Freedom Mode
- Interactive canvas with 22 draggable stars
- Drag stars close together to connect them with glowing lines
- Connect enough → triggers "Ouma 💜" constellation message
- "you light up my night. not even joking"
- Floating hearts spawn on trigger

### 😂 Chaos Button
- "DO NOT PRESS (3wiwra edition)"
- Screen flashes, cars fly everywhere
- Goofy Darija messages
- Web Audio API comedic tones
- 5-second chaos sequence

### 🎨 Micro-Details
- **Glowing purple cursor** follows mouse everywhere
- **Twinkling animated stars** in every section
- **Nav dots** (right side) — click to jump between sections, highlight active section
- **Rain toggle 🌧️** — top right button, activates purple rain overlay
- **Ambient night sound** — subtle low-pass white noise
- **Page title rotates** between cute messages
- **Scroll fade-in animations** for sections
- **Bounce physics** on hover for all interactive elements
- **Mobile responsive** — works on phones

## 🛠️ Tech Stack

- Pure HTML5 / CSS3 / Vanilla JavaScript (no frameworks)
- Google Fonts: Quicksand + Pacifico
- Web Audio API (ambient + chaos sounds)
- Canvas 2D API (star constellation)
- IntersectionObserver API (scroll effects + nav dots)
